## Overview
**Gamers Shop** is a front-end e-commerce website focused on selling **PC and console games**. The project is developed entirely on the **front-end side** using modern React technologies and does not include a backend or real API integration at its current stage.

The website is designed in **English** and follows a **Dark Mode** theme to align with the visual preferences commonly associated with gaming platforms.

This project is intended as a **portfolio project** and is actively under development, with multiple improvements planned for future versions.

## Website Structure

The initial structure of the website consists of the following main sections:

### Home
The landing page of the website, providing a general introduction to the store and easy access to other sections. It serves as the entry point for users and highlights the overall concept of the platform.

### Shop
The main store section where PC and console games are displayed using product cards.  
Currently, all product data is **static** and used only for UI demonstration purposes.

### About
This section provides general information about the Gamers Shop project, its purpose, and the concept behind the website.

### Sign In / Sign Up
Authentication-related pages designed purely at the **UI level**.  
No real authentication logic, backend, or user management system is implemented yet.

### Cart
The shopping cart section is visually implemented but currently **static**.  
It does not include real-time product management, price calculations, or backend connectivity.

## Current Limitations
- No backend or database integration
- Static product data
- Shopping cart is not dynamic
- Authentication is UI-only
- Images are **not optimized**, resulting in slower load times
- Performance and SEO optimizations have not been applied yet
